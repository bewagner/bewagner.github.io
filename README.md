# bewagner.gitub.io

This is the code for my personal website.
It is based on the [Zolarwind](https://github.com/thomasweitzel/zolarwind) Zola template.

## Setup
I use pixi to manage the Zola dependency. So first run.
```
pixi install
pixi shell
```

Then run 
```
zola serve --drafts
```
to serve the page locally.
